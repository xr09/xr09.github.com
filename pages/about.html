About

I like to think of myself as a software developer, simplicity lover, architect padawan and performance obsessed.

I've been working with Python since 2007, mostly PyQt and CLI apps, some PHP with Symfony, and lately lots of shell scripting.

I'm currently using Symfony at work, but Django and Flask for personal projects.

On the server side I like to use nginx, uwsgi, php-fpm, postgresql, dnsmasq, memcache, varnish and others.

For my personal development workflow love the simplicity of mercurial but use git too for github of course.
